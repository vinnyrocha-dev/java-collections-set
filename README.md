# Java Collections - Set

Este repositório contém exemplos práticos de utilização da interface **Set** em Java utilizando diferentes implementações da coleção.

## 📚 Conteúdo

Os exemplos demonstram o funcionamento das principais implementações de Set:

- HashSet
- LinkedHashSet
- TreeSet

## 📂 Exemplos incluídos

### HashSet
- Estrutura que **não mantém ordem dos elementos**
- **Não permite duplicação**
- Possui **acesso mais rápido aos dados**

Classes:
- ExemploHashSet
- ExemploHashSetAluno

### LinkedHashSet
- Mantém a **ordem de inserção dos elementos**
- Não permite duplicados

Classe:
- ExemploLinkedHashSet

### TreeSet
- Mantém os elementos **ordenados automaticamente**
- Não permite duplicação
- Utiliza **ordem natural ou Comparable**

Classes:
- ExemploTreeSet
- ExemploTreeSetAluno

## 🧠 Conceitos praticados

- Java Collections Framework
- Interface Set
- HashSet
- LinkedHashSet
- TreeSet
- Comparable
- equals() e hashCode()
- Manipulação de objetos em coleções
